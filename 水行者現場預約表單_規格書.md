## 一、專案概述

### 1.1 目的

為水行者海洋運動中心建立一個一頁式現場預約表單，讓客戶可以線上選擇日期時段、挑選並混搭多個活動項目、指定人數，系統即時計算總價，客戶送出後系統將訂單自動推播到公司 LINE 群組，匯款後由客服人工對帳。

### 1.2 基本資訊

| 項目 | 說明 |
|------|------|
| 表單類型 | 一頁式（Single Page Form） |
| 付款方式 | 匯款（不刷卡） |
| 通知方式 | LINE Messaging API 推播到指定群組 |
| 語言 | 繁體中文 |
| 目標裝置 | Mobile First（手機優先），同時支援桌機 |
| 嵌入方式 | iframe 嵌入 waterman-sports.com 既有頁面 |

### 1.3 使用流程

```
客戶進入表單
  ↓
第 1 步：選擇日期（系統自動判斷平假日）＋ 時段
  ↓
第 2 步：加入活動項目到購物車（可混搭多個）
         ├─ SUP 一人一版：指定大人/兒童人數
         ├─ SUP 兩人一版：指定版數 + 兒童替換數
         └─ 獨木舟：指定船數 + 中座兒童加購數
  ↓
即時顯示各項目金額與訂單小計
  ↓
第 3 步：勾選秘境保證（選填，全單 × 1.2 倍）
  ↓
第 4 步：填寫聯絡資料（姓名/電話/Email/LINE ID）
  ↓
送出訂單
  ↓
顯示匯款資訊 + 訂單編號
  ↓
系統同步推播到 LINE 群組
  ↓
客戶匯款後以 LINE 聯絡客服提供末 5 碼
  ↓
客服核對後標記訂單為已付款
```

## 二、活動項目與價格結構

### 2.1 活動項目

| 項目代碼 | 名稱 | 船板類型 | 成團人數 |
|---------|------|---------|---------|
| `sup_solo` | 豆腐岬 SUP 一人一版 | 1 人 1 版 | 4 人（訂單總人數） |
| `sup_duo` | 豆腐岬 SUP 兩人一版 | 2 人 1 版 | 4 人（訂單總人數） |
| `kayak` | 豆腐岬 獨木舟兩人一船 | 2 人 1 船 | **4 人（獨木舟本身必須 ≥ 4 人才能開團）** |

### 2.2 基礎單價表

| 項目 | 平日 | 假日 | 計價單位 |
|------|------|------|---------|
| SUP 一人一版 | 1,200 | 1,500 | 每人 |
| SUP 兩人一版 | 2,100 | 2,700 | 每版（2 位大人） |
| 獨木舟兩人一船 | 800 | 1,000 | 每「大人座」 |

### 2.3 兩人一版計算邏輯

- 平日整版價 = 第一人 1,200 + 第二人 900 = **2,100**
- 假日整版價 = 第一人 1,500 + 第二人 1,200 = **2,700**

### 2.4 兒童規則

| 項目 | 年齡區間 | 規則 | 金額 |
|------|---------|------|------|
| SUP 一人一版 | 6~12 歲 | 每位兒童 **減 500**，須大人陪同 | `-500 × 兒童人數` |
| SUP 兩人一版 | 6~12 歲 | 每版預設 2 位大人；替換為兒童者 **每位減 200**，不可超過版數；須大人陪同 | `-200 × 替換兒童數` |
| 獨木舟 | 4~6 歲 | **坐中間不持槳**，以加購形式，**每位加 500**；不佔大人座；不可超過船數 | `+500 × 中座兒童數` |

> ⚠️ 獨木舟 6 歲以上視為大人（佔大人座位）；4~6 歲兒童僅能坐中間加購座位。

### 2.5 秘境保證規則

- 勾選後**訂單小計 × 1.2 倍**（加價 20%）
- **作用於訂單層級**，所有活動項目同步套用，不是個別項目覆蓋單價
- 加價金額 = `baseTotal × 0.2`（四捨五入取整）
- 表單必須顯示警語：

> ⚠️ 秘境保證依當天風浪狀況和能見度決定是否抵達或執行，如無法抵達則退秘境保證之差價。

### 2.6 活動贈送服務（顯示於表單說明區）

- 單團 6 人以上贈空拍
- 豆腐岬標準下水時間：90~100 分鐘
- 秘境保證（賊仔澳）下水時間：120~150 分鐘

## 三、平假日判斷邏輯

### 3.1 規則

```
是否為假日 = (星期六 OR 星期日 OR 國定假日)
是否適用假日價 = 是否為假日 AND (月份 ∈ 5~10 月)
```

| 日期情境 | 套用費率 |
|---------|---------|
| 平日（週一至週五非國定假日） | 平日價 |
| 假日 + 月份 11~4 月 | **平日價**（淡季優惠） |
| 假日 + 月份 5~10 月 | 假日價（旺季原價） |
| 國定假日（5~10 月） | 視為假日 |

### 3.2 演算法範例

```ts
export function shouldUseHolidayRate(date: Date): boolean {
  const day = date.getDay();                  // 0=日, 6=六
  const month = date.getMonth() + 1;          // 1~12
  const isWeekend = day === 0 || day === 6;
  const isPublicHoliday = checkPublicHoliday(date); // 經 date.nager.at 快取
  const isHoliday = isWeekend || isPublicHoliday;
  return isHoliday && month >= 5 && month <= 10;
}
```

### 3.3 國定假日資料來源

- 台灣政府開放資料：https://data.gov.tw/dataset/14718
- 第三方 API：https://date.nager.at/api/v3/publicholidays/{year}/TW
- 建議做本地快取，每年 12 月更新一次

## 六、價格計算邏輯

### 6.1 型別定義

```ts
type Activity = "sup_solo" | "sup_duo" | "kayak";

type BookingItem =
  | { id: string; activity: "sup_solo"; adults: number; children: number }
  | { id: string; activity: "sup_duo"; boards: number; children: number }
  | { id: string; activity: "kayak";   boards: number; midKids: number };

interface ItemPriceBreakdown {
  id: string;
  activity: Activity;
  unitPrice: number;
  subtotal: number;
  discount: number;
  surcharge: number;
  total: number;       // subtotal - discount + surcharge
  peopleCount: number; // 該項目算入訂單的人頭數
}

interface OrderPriceBreakdown {
  items: ItemPriceBreakdown[];
  baseTotal: number;       // Σ item.total
  secretSurcharge: number; // isSecret ? round(baseTotal * 0.2) : 0
  grandTotal: number;      // baseTotal + secretSurcharge
  dayType: "weekday" | "holiday";
  totalPeople: number;     // Σ item.peopleCount
  kayakPeople: number;     // 獨木舟項目人頭總和（用於 4 人門檻檢查）
}
```

### 6.2 常數

```ts
const UNIT_PRICE = {
  sup_solo: { weekday: 1200, holiday: 1500 },
  sup_duo:  { weekday: 2100, holiday: 2700 },
  kayak:    { weekday:  800, holiday: 1000 },
};

const SUP_SOLO_CHILD_DISCOUNT = 500;
const SUP_DUO_CHILD_DISCOUNT  = 200;
const KAYAK_MID_KID_SURCHARGE = 500;
const SECRET_MULTIPLIER       = 1.2;
```

### 6.3 單項計算

```ts
function calculateItemPrice(item: BookingItem, dayType: DayType): ItemPriceBreakdown {
  const unitPrice = UNIT_PRICE[item.activity][dayType];
  let subtotal = 0, discount = 0, surcharge = 0, peopleCount = 0;

  if (item.activity === "sup_solo") {
    peopleCount = item.adults + item.children;
    subtotal    = unitPrice * peopleCount;
    discount    = SUP_SOLO_CHILD_DISCOUNT * item.children;
  } else if (item.activity === "sup_duo") {
    peopleCount = item.boards * 2;
    subtotal    = unitPrice * item.boards;
    discount    = SUP_DUO_CHILD_DISCOUNT * item.children;
  } else {
    // kayak
    const adultSeats = item.boards * 2;
    peopleCount = adultSeats + item.midKids;
    subtotal    = unitPrice * adultSeats;
    surcharge   = KAYAK_MID_KID_SURCHARGE * item.midKids;
  }

  return {
    id: item.id,
    activity: item.activity,
    unitPrice,
    subtotal,
    discount,
    surcharge,
    total: subtotal - discount + surcharge,
    peopleCount,
  };
}
```

### 6.4 訂單彙總

```ts
function calculateOrderPrice(input: { items: BookingItem[]; bookingDate: string; isSecret: boolean }): OrderPriceBreakdown {
  const dayType = shouldUseHolidayRate(new Date(input.bookingDate)) ? "holiday" : "weekday";
  const items = input.items.map(it => calculateItemPrice(it, dayType));
  const baseTotal = items.reduce((s, r) => s + r.total, 0);
  const secretSurcharge = input.isSecret ? Math.round(baseTotal * (SECRET_MULTIPLIER - 1)) : 0;
  const grandTotal = baseTotal + secretSurcharge;
  const totalPeople = items.reduce((s, r) => s + r.peopleCount, 0);
  const kayakPeople = items.filter(r => r.activity === "kayak").reduce((s, r) => s + r.peopleCount, 0);
  return { items, baseTotal, secretSurcharge, grandTotal, dayType, totalPeople, kayakPeople };
}
```

### 6.5 計算範例

#### 範例 A：平日、SUP 一人一版（2 大 1 小）＋ 獨木舟 2 艘（1 中座兒童）

- `sup_solo`：1,200 × 3 − 500 = **3,100**（3 人）
- `kayak`：800 × 4 + 500 × 1 = **3,700**（5 人）
- `baseTotal` = 6,800
- `secretSurcharge` = 0
- **grandTotal = 6,800**，totalPeople = 8，kayakPeople = 5

#### 範例 B：假日（7 月）、SUP 兩人一版 2 版（1 位兒童）＋ 秘境保證

- `sup_duo`：2,700 × 2 − 200 × 1 = **5,200**（4 人）
- `baseTotal` = 5,200
- `secretSurcharge` = round(5,200 × 0.2) = **1,040**
- **grandTotal = 6,240**，totalPeople = 4

#### 範例 C：假日（12 月，淡季）、獨木舟 1 艘（2 大）

- 套用平日價
- `kayak`：800 × 2 = **1,600**（2 人）
- `baseTotal` = 1,600，grandTotal = 1,600
- ⚠️ 獨木舟人數 = 2 < 4，顯示未達開團提醒

### 10.2 通訊協議

POST JSON 至 Apps Script Web App：

```ts
// Request
{
  bookingDate: "2026-04-25",
  timeSlot: "morning",
  isSecret: false,
  items: BookingItem[],
  customerName: string,
  customerPhone: string,
  customerEmail: string,
  customerLineId: string,
  notes?: string,
}

// Response
{
  ok: true,
  result: {
    orderNo: "WM-20260425-0003",
    bankInfo: { bankName, branch, accountName, accountNumber },
    customerServiceLineId: string,
  }
}
```
